# Инструкция по добавлению иконки приложения

## Проблема: Нет иконки у приложения

Приложение запускается без иконки, потому что файлы иконок отсутствуют в Assets.xcassets.

## Решение: Добавить иконку через Xcode

### Способ 1: Использовать генератор иконок Xcode (Рекомендуется)

1. Откройте проект в Xcode:
   ```bash
   open Proskomidiya.xcodeproj
   ```

2. В навигаторе проекта выберите:
   - `Proskomidiya` → `Proskomidiya` → `Assets.xcassets` → `AppIcon`

3. В центральной панели вы увидите сетку размеров иконок

4. **Добавьте иконку 1024x1024:**
   - Найдите секцию "iOS Marketing" (1024 x 1024)
   - Перетащите изображение 1024x1024 пикселей в это поле
   - Xcode автоматически сгенерирует все необходимые размеры

5. Если у вас нет готовой иконки, создайте простую временную:
   - Можно использовать любую картинку 1024x1024
   - Или создать простую иконку с текстом "П" (Проскомидия)

### Способ 2: Использовать онлайн генератор

1. Подготовьте изображение 1024x1024 пикселей

2. Используйте онлайн генератор:
   - https://www.appicon.co
   - https://appicon.build
   - Загрузите ваше изображение
   - Скачайте сгенерированный набор иконок

3. В Xcode:
   - Выберите `AppIcon` в Assets.xcassets
   - Перетащите каждую иконку в соответствующий размер

### Способ 3: Создать простую временную иконку

Если нужно быстро запустить приложение для тестирования:

1. Создайте простой PNG файл 1024x1024 пикселей с:
   - Золотым фоном (#BC9E3F)
   - Белым текстом "П" или крестом в центре

2. Добавьте его через Xcode как описано в Способе 1

## Проверка

После добавления иконки:

1. Очистите сборку: `Cmd + Shift + K`
2. Пересоберите проект: `Cmd + B`
3. Запустите приложение: `Cmd + R`
4. На главном экране эмулятора/устройства должна появиться иконка приложения

## Размеры иконок для iOS

Минимально необходимые размеры:
- **1024x1024** - для App Store и главного экрана (iOS Marketing)

Рекомендуемые для полной поддержки:
- 20x20 (@2x, @3x) - для уведомлений
- 29x29 (@2x, @3x) - для настроек
- 40x40 (@2x, @3x) - для Spotlight
- 60x60 (@2x, @3x) - для главного экрана iPhone
- 76x76 (@1x, @2x) - для главного экрана iPad
- 83.5x83.5 (@2x) - для iPad Pro
- 1024x1024 - для App Store

## Советы

- Используйте PNG формат без прозрачности (фон должен быть непрозрачным)
- Избегайте текста на иконке (особенно маленького)
- Иконка должна хорошо смотреться в маленьком размере
- Используйте простые формы и контрастные цвета

## Быстрое решение для тестирования

Если нужно срочно протестировать приложение без иконки:

1. В Xcode выберите `AppIcon` в Assets.xcassets
2. Найдите секцию "1024 x 1024 (iOS Marketing)"
3. Перетащите ЛЮБОЕ изображение (даже скриншот) - это будет временная иконка

**Примечание:** Для публикации в App Store потребуется качественная иконка 1024x1024.
