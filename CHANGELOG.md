## Функциональность

### Офлайн режим
- Приложение сначала пытается загрузить локальный HTML файл
- При отсутствии локального файла загружается веб-сайт из интернета
- При ошибках сети автоматически переключается на локальный HTML

### Навигация
- Кнопки "Назад" и "Вперёд" для навигации по истории
- Кнопка "Обновить" для перезагрузки страницы
- Кнопка "Главная" для возврата на главную страницу
- Прогресс-бар показывает процесс загрузки

### Безопасность
- Настроена поддержка App Transport Security
- Ограничен доступ к определённым доменам
- Безопасная обработка внешних ссылок

## Технические детали

- Минимальная версия iOS: 15.0
- Архитектуры: armv7, arm64
- Язык: Swift 5
- Интерфейс: UIKit + Storyboard
- WebView: WKWebView
