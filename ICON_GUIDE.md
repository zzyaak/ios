# Руководство по созданию иконок для iOS

## Требования к иконкам

### Основная иконка приложения (1024x1024)
- **Размер**: 1024x1024 пикселей
- **Формат**: PNG
- **Прозрачность**: Без прозрачности
- **Цветовое пространство**: RGB

### Размеры для разных устройств

#### iPhone
- **20x20** (@1x, @2x, @3x) → 20, 40, 60 пикселей
- **29x29** (@1x, @2x, @3x) → 29, 58, 87 пикселей  
- **40x40** (@1x, @2x, @3x) → 40, 80, 120 пикселей
- **60x60** (@2x, @3x) → 120, 180 пикселей

#### iPad
- **20x20** (@1x, @2x) → 20, 40 пикселей
- **29x29** (@1x, @2x) → 29, 58 пикселей
- **40x40** (@1x, @2x) → 40, 80 пикселей
- **76x76** (@1x, @2x) → 76, 152 пикселей
- **83.5x83.5** (@2x) → 167 пикселей

## Дизайн иконки

### Рекомендуемый стиль для "Проскомидия"

**Цветовая схема**:
- Основной цвет: #8B4513 (коричневый)
- Акцентный цвет: #DAA520 (золотой)
- Фон: #F0EBE0 (кремовый)

**Элементы дизайна**:
- Крест или религиозный символ
- Стилизованные элементы монастыря
- Простой и узнаваемый силуэт

**Примеры элементов**:
- Православный крест
- Купол церкви
- Стилизованная икона
- Монастырские ворота

### Технические требования

1. **Простота**: Иконка должна быть узнаваемой в маленьком размере
2. **Контраст**: Хорошая видимость на разных фонах
3. **Уникальность**: Отличаться от других религиозных приложений
4. **Соответствие**: Следовать принципам iOS Human Interface Guidelines

## Пошаговое создание

### 1. Создание основной иконки (1024x1024)

1. Создайте квадратный холст 1024x1024 пикселей
2. Установите фоновый цвет #8B4513
3. Добавьте основной элемент (крест, купол)
4. Используйте золотой цвет #DAA520 для акцентов
5. Убедитесь, что иконка хорошо смотрится в квадрате

### 2. Создание всех размеров

Используйте генератор иконок или создайте вручную:

```
1024x1024 → App Store
180x180 → iPhone @3x
120x120 → iPhone @2x
167x167 → iPad Pro @2x
152x152 → iPad @2x
120x120 → iPhone @2x
87x87 → iPhone @3x
80x80 → iPad @2x
76x76 → iPad @1x
60x60 → iPhone @1x
58x58 → iPhone @2x
40x40 → iPad @1x
29x29 → iPhone @1x
20x20 → iPhone @1x
```

### 3. Тестирование

1. **Разные размеры**: Проверьте читаемость в маленьких размерах
2. **Разные фоны**: Белый, черный, цветной
3. **Разные устройства**: iPhone, iPad
4. **Разные разрешения**: Retina, обычные экраны

## Загрузка в проект

### 1. Подготовка файлов

Создайте папку с иконками:
```
AppIcon/
├── AppIcon-1024.png
├── AppIcon-180.png
├── AppIcon-120.png
├── AppIcon-167.png
├── AppIcon-152.png
├── AppIcon-87.png
├── AppIcon-80.png
├── AppIcon-76.png
├── AppIcon-60.png
├── AppIcon-58.png
├── AppIcon-40.png
├── AppIcon-29.png
└── AppIcon-20.png
```

### 2. Добавление в Xcode

1. Откройте `Assets.xcassets`
2. Выберите `AppIcon`
3. Перетащите файлы в соответствующие места
4. Проверьте, что все размеры заполнены

### 3. Проверка

1. Запустите приложение на симуляторе
2. Проверьте иконку на домашнем экране
3. Убедитесь, что иконка корректно отображается

2. **Используйте векторную графику**: Для четкости в любом размере
3. **Тестируйте на устройствах**: Проверьте на реальных iPhone и iPad
4. **Следуйте гайдлайнам**: Изучите Apple Human Interface Guidelines
5. **Уникальность**: Создайте узнаваемую иконку
